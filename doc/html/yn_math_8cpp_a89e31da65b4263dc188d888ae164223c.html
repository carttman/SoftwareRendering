B3YXVec2CCW : 2성분 외적, 외적의 결과가 양수면, 'CCW' .Back-Face Culling 에 사용. : D3DXVec2CCW 대응 (https://learn.microsoft.com/en-us/windows/win32/direct3d9/d3dxvec2ccw)

2성분으로는 구조적으로 외적이 불가능하므로 3차원으로 확장 사용합니다. v0=(x1,y1,0) cross v1=(x2,y2,0). 또는 아래 수식을 참조하십시오.

v.y = v0.z * v1.x - v0.x * v1.z;
v.z = v0.x * v1.y - v0.y * v1.x;
