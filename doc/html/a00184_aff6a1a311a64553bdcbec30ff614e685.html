뷰포트 멥핑 (Viewport)

정규장치공간(NDC) 의 정점을 출력 화면 좌표(Screen) 로 변환합니다.
입력된 좌표는 해상도에 맞게 보정(Scale & Bias) 되어 다음 파이프라인으로 전달됩니다.

화면 좌표로 변환 : 해상도 (800x600) 기준
pos.x = -1 ~ 1 --> 0 ~ 799
pos.y = -1 ~ 1 --> 0 ~ 599
