Yena GUID 구조체

Windows / GUID 와 동일한 128bit 데이터 구조를 유지합니다.
GUID 는 16진수로 설정된 내부 데이터의 묶음으로 고유한 식별값을 구성합니다.
Yena GUID "YNID" 도 비슷한 (하지만 간편한) 방식으로 고유 ID 를 설정하겠습니다.

};
