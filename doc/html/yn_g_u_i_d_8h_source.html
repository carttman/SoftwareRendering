42//! ### GUID/UUID 정의
43//! Windows COM 은 인터페이스 및 클래스 객체 식별용 ID 로 GUID(UUID)를 사용합니다.
44//! GUID 는 128bit 데이터 구조체이며 보통 문자열 형태로 정의됩니다.
45//! Windows COM 의 GUID 들은 별도 정의되어 라이브러리(UUID.lib) 형태로 배포됩니다.
46//! \code
47//! < 표준 GUID >
48//! IUnknown = {{16진수 8개}{16진수 4개}{16진수4개}{16진수4개}{16진수12개}}
49//! = "00000000-0000-0000-C000-000000000046"
50//! \endcode
51//!
52//! 사용자 정의 GUID 생성은 guidgen.exe 또는 cuidgen.exe 같은 단독 툴이나 또는
53//! CoCreateGUID, UUidCreate 함수를 사용합니다.
54//!
55//! ### Yena GUID 설정
56//! SWR 는 정규 GUID 형식에서 살짝 벗어나 적당한 IID (임의의 상수)를 정의하겠습니다.
