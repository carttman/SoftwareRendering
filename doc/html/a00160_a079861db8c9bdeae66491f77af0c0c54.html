렌더링 작업을 시작합니다.

이 메소드의 호출 이전에 정점버퍼, FVF, 마디 크기(Stride) 등 렌더링에 필요한 모든 정보가 디바이스에 등록되어 있어야 합니다.

Parameters
PrimitiveType렌더링 기하 타입.
StartVertex렌더링 정점 시작번호 (정점 버퍼 안)
PrimitiveCount렌더링 기하 개수.
Returns
성공시 OK, 실패시 FALSE
Note
<Yena> 주요 렌더링 작업은 Yena GraphicsEngine 에서 처리합니다. (v1.5.3 이상)
