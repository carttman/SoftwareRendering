들어가기

Yena SWR 는 CPU 기반 소프트웨어 렌더링 엔진입니다.
Yena 의 중요 목표중 하나는 현대의 하드웨어(GPU) 가 수행하는 가장 일반화된 렌더링 파이프라인을 준수하는 SW 장치를 개발하고 3D 렌더링에 수반되는 여러 학술적 연구를 구체화하는 것입니다.

전체 개발의 기준점은 Direc3D9 의 인터페이스입니다. 시대적으로는 이미 "클래식" 범주에 들어가지만 여타의 현대적 그래픽스 API (DX11/12, Vulkan) 에 비하여 직관적인 파이프라인과 상대적으로 평이한 API 의 운용성으로 우리가 만들 SW 렌더링 엔진의 기준점이 되기에는 모자람이 없습니다.


우리가 할 일

전체 학습 과정은 다음을 따라가겠습니다.

  • 컴퓨터 그래픽스 기초
  • 3D 프로그래밍 기초
  • 렌더링 장치 구축
  • 3D 기하와 변환
  • 렌더링 파이프라인 구축 : 정점,기하,픽셀 파이프라인, ...
  • 셰이더 파이프라인 구축 : GPU 구조의 이해, HLSL 기초, ...
  • 조명과 텍스처
  • 다중 텍스처 멥핑
  • 이미지 기반 렌더링 기초
  • [부록1] COM 과 Interface
  • [부록2] 3D 수학

학습 진도별 렌더링 기술을 연구하고, 예제 및 그래픽엔진("Yena") 에 적용하여 구조와 성능을 개량해 나갑니다.
각 예제별 도움말과 온라인 기술문서를 통해 DX (하드웨어 가속 API) 의 구조도 함께 살펴보겠습니다.

상기 거론된 주제 이외에 필요한 3D 기술도 가능한 Yena 에서 구현하도록 하겠습니다.
다만, SW 렌더링 특성상 GPU 가속없이 HW 의 물리적 기능을 온전히 구현할 수 없는 경우에는 적정선에서 타협하겠습니다.


우리의 기대

Yena 를 통해 수련된 3D 렌더링 기술과 파이프라인 운용의 이해도는 추후 하드웨어 가속 3D 렌더링 API (DX11/12, Vulkan) 학습시 그 기술적 허들을 최소화 해 줄 것입니다.
