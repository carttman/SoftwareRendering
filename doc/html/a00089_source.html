75//! \par 정점버퍼
76//! 정점버퍼는 일련의 기하(Geometry)를 구성하는 정점 데이터가 저장되는 버퍼입니다.
77//! 이 데이터들은 정점 단위규격(Vertex Format 또는 Vertex Declaration) 으로 배열화 저장되며 읽기 전용으로 GPU 에 전달됩니다.
78//! 필요시 동적 정점 버퍼 (Dynamic Vertex Buffer) 를 구성하는 것도 가능합니다.
79//!
80//! \par 정점버퍼 렌더링
81//! 렌더링 장치는 렌더링 파이프라인에 설정된 정점버퍼를 읽어오며, 렌더링 기하구성(Primitive Topology)
82//! 주문에 따라 데이터를 처리합니다. (Vertex Streaming)
83//! 렌더링 파이프라인은 하나 이상의 정점버퍼를 연결(Binding) 할 수 있으며 (DX9, max.16)
84//! 정점 조립 스테이지(Vertex Assembly 또는 Input Assembly Stage) 에서 재구성되어 사용됩니다.
85//!
86//! <DX9> IDirect3DVertexBuffer9 으로 정점버퍼를 운용합니다.
87//! <Yena> IYenaVertexBuffer9 으로 DX 인터페이스를 대응합니다.
88//!
89//! \note 장치(Device)는 파이프라인에 설정된 "현재" 정점 버퍼를 사용하므로 렌더링 명령
90//! (Draw / DrawPrimitive) 전에 장치에 연결(Binding) 되어야 합니다.
91//!
92//! \see B3YenaVertexBuffer9
93//! \see IYenaVertexBuffer9
94//! \see [IDirect3DVertexBuffer9](https://learn.microsoft.com/en-us/windows/win32/api/d3d9/nn-d3d9-idirect3dvertexbuffer9)
