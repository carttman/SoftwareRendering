#define topic_Shader_x

셰이더는 사용자 정의 정점 연산을 수행하며 GPU 언어로 제작됩니다.
전문 그래픽 API 들은 각각의 목적에 맞추어 GPU 전용 함수(명령어)와 자료형을 제공합니다.
해당 언어로 작성된 소스는 GPU 용 컴파일러를 통해 바이너리화 하고, 전용 인터페이스로 GPU 에 설정합니다. 

  <DX9> 셰이더 인터페이스  
  IDirect3DVertexShader9  
  IDirect3DPixelShader9

다른 GPU / 장치 설정과 마찬가지로 한번에 파이프라인 기능별 1개의 셰이더를 장치에 등록할 수 있으며 GPU 내의 TnL 파이프라인 (내장) 기능을 대신합니다.

셰이더 연산에 사용되는 행렬, 조명, 재질 등의 셰이더 상수은 사용자가 자유롭게 정의 할 수 있지만 GPU 내 고정함수 파이프라인과의 연계성, 레지스터 호환성 등 물리적 제한에 유의해야 합니다.
셰이더 상수들은 전용 인터페이스로 사용자가 직접 GPU에 설정해야 합니다.

Note
GPU/장치는 리셋 되거나 다른 셰이더로 대체될 때까지 해당 셰이더, 기능 및 상태를 유지합니다.

<DX9> 셰이더 모델(Shader Model, SM) 3.0 기반 고수준 셰이더 언어(HLSL)을 지원합니다.
<DX11> SM5.x 지원
<DX12> SM6.x 지원
<Yena> SM3 및 HLSL 을 최대한 구현합니다.
