## Yena 정점색 처리 과정 

* 정점색 = (100% 빨강, 0% 초록, 50% 파랑)
```
 [ 정점 버퍼 구성]                        [ Yena 로딩 후 ]                           [ 출력 "렌더타겟" ]
 [ CPU / VB / Color / DWORD / DX ]  --> [ Yena / Color / float / B3YXCOLOR ]  --> [ RT / Color / DWORD / GDI ]
      0x AA RR GG BB                -->        BYXCOLOR(R, G, B, A)           -->       0x AA BB GG RR
     (0x ff ff  00  80 )            -->          (1.0, 0, 0.5, 1)             -->     (0x ff 80  00  ff )
```
