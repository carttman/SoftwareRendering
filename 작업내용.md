## 프로젝트 개요

* 기본 구성  
  - Windows Desktop Application
  - 32/64비트 지원
  - 유니코드 버전 (UTF-8)  

* 예제 소스 구성
  ```c
  main.cpp              //애플리케이션 설정 및 운용 소스
  Device.cpp            //렌더링 장치 운용 소스 (각 예제별 개정)
  Render.cpp            //렌더링 데모 소스 (각 예제별 개정)
  ```

* Yena SWR 구성
  ```c
  Yena.h/cpp            //Yena SWR 메인 클래스 정의 (DX9 대응)
  ynGUID.h/cpp			//Yena COM GUID 정의
  ynx9.h                //Yena SWR 인터페이스 정의 (DX9 대응)
  ynx9types.h           //Yena SWR 자료형 및 상수 정의 (DX9 대응)
  ynError.h/cpp         //예외처리, 에러, 로그 출력
  ynMath.h/cpp          //수학 자료형 및 헬퍼 정의
  ```
