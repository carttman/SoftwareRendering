## 예제 내용 

* 과제 제작용
  - 그리기 함수 GDI 사용.
  - 사용자 그리기 함수로 변경.(과제)★

* 렌더링 파이프라인 구축
  - 2D 정점 운용, 변환 없음
  - 정점 버퍼 생성 및 운용
  - 정점 구조(FVF) 설정
  - 렌더링 상태 조절 
  - 렌더링 API 추가 및 개정

* 정점색 적용 (과제) ★ 
  - 정점 구조(FVF) 변경
  - 정점색 추가 (DWORD) 
  - 정점색 추가 (B3YXCOLOR) ★
  - 색상 구조체/클래스 연산자 추가
  - B3YXCOLOR 와 DWORD 호환성 증대 (+연산자추가) ★

* 개정 내역 
  ```
  struct COLVXT   : 색상 추가
  struct B3YCOLOR : 연산자 추가
  struct B3YXCOLOR : 연산자 추가
  ynMath
  IYenaDevice9
  ```

* Yena 개정 (25.0619)
  - Yena 운용 함수명 접두어 변경
  - 구형 호환성 유지용 재정의 포함
  ```
  (구) YnSort
  (신) ynSort
  #define YnSort ynSort
  ```

## 프로젝트 개요

* 기본 구성  
  - Windows Desktop Application
  - 32/64비트 지원
  - 유니코드 버전 (UTF-8)  

* 예제 소스 구성
  ```c
  main.cpp              //애플리케이션 설정 및 운용 소스
  Device.cpp            //렌더링 장치 운용 소스 (각 예제별 개정)
  Render.cpp            //렌더링 데모 소스 (각 예제별 개정)
  ```

* Yena SWR 구성
  ```c
  Yena.h/cpp            //Yena SWR 메인 클래스 정의 (DX9 대응)
  ynGUID.h/cpp			//Yena COM GUID 정의
  ynx9.h                //Yena SWR 인터페이스 정의 (DX9 대응)
  ynx9types.h           //Yena SWR 자료형 및 상수 정의 (DX9 대응)
  ynError.h/cpp         //예외처리, 에러, 로그 출력
  ynMath.h/cpp          //수학 자료형 및 헬퍼 정의
  ```
